Visão geral do projeto

Este projeto foi realizado no âmbito do curso Scrimba, no qual tive de simular chamadas a APIs, começando por usar Mirage JS para simular um servidor externo. Mais tarde, passei a utilizar uma base de dados Firestore. Nesta aplicação, usei React.js com a biblioteca React Router. A realização deste projeto proporcionou-me conhecimentos aprofundados sobre o React Router e a reutilização de componentes.

A aplicação consiste no aluguer de caravanas para viagem e contém uma página onde os hosts conseguem fazer login para consultar as suas publicações de aluguer. (Se quiser testar, os dados de login são: email — b@b.com | password — p123)
